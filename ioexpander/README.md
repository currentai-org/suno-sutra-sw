This is a Circuit Python project for a Seeeduino XIAO RP2350 microcontroller.
Requires CircuitPython v10.x. Tested on v10.0.3.
https://circuitpython.org/board/seeeduino_xiao_rp2350/

After flashing with circuitpython, simply copy the contents of this folder onto the flash storage device that appears when the RP2350 is connected to the PC.
